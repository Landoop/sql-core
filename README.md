# sql-core
The basic lib supporting our json-sql, avro-sql and kafka-connect-sql-smt

**2.0 (2020-1-1)**

* Update to scala 2.12

**0.1 (2017-04-18)**

* first release

### Building

***Requires gradle 5.0 to build.***

To build

```bash
gradle compile
```

To test

```bash
gradle test
```


You can also use the gradle wrapper

```
./gradlew build
```
